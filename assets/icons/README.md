# Local application icons

These original icons were created for Open Posture and are covered by the repository's Apache-2.0 license. `trayTemplate` uses a monochrome macOS template form; the PNG and ICO derivatives are generated from the checked-in SVG sources. The macOS `.icns` is generated from `assets/packaging/OpenPosture-mac.svg`, whose full-bleed background prevents macOS from wrapping the artwork in a second white legacy-icon tile.
